# Document Prototype Design Pattern

## 📌 Overview
This project demonstrates the **Prototype Design Pattern** applied to a 
document management system. Instead of creating new document objects from 
scratch, this pattern clones pre-configured prototype instances through a 
central registry — making object creation faster and more consistent.

---

## 🧩 Pattern Used
**Creational Design Pattern → Prototype Pattern**

The Prototype Pattern is used when the cost of creating a new object is 
more expensive than copying an existing one. A prototype object is stored, 
and whenever a new instance is needed, it is cloned from that prototype.
